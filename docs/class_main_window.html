Detailed Description

Main window of the application.

Has two rows of widgets, one for devices and one for scan-parameter-selections which are used to apply parameter values and loop over them later, for example if you want to ramp the voltage over time. Also has a menu to get to the special functions like settings, file selection or the serial console. Buttons to start or stop a measurement are also here.
